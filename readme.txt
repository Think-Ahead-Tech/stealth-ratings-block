Gather ratings from your site visitors, quietly and efficiently, with a customizable Gutenberg block.

== Description ==

Stealth Ratings Block lets you easily collect feedback or ratings from your website visitors using a simple, privacy-conscious block. Designed for seamless integration with the WordPress block editor (Gutenberg), this plugin helps you gain valuable insights without disrupting the user experience.

**Features:**
* Add a customizable ratings block to any post or page
* Collect feedback or star ratings from visitors
* Send feedback to a designated email address
* Lightweight and privacy-friendly â€“ no tracking or third-party scripts
* Easy setup, no coding required

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/stealth-ratings-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Add the "Stealth Ratings" block to any post or page using the block editor.
4. Configure settings via Settings > Stealth Ratings in the WordPress admin.

== Frequently Asked Questions ==

= Where can I find the ratings submissions? =
Feedback can be sent to your specified email address, set in the plugin settings.

= Is any visitor data tracked or shared? =
No. Stealth Ratings Block is designed to be privacy-friendly and does not track users or share data with third parties.

= Can I customize the appearance of the block? =
Yes, the block supports basic customization options in the editor.
